Prayer Times API

Description

This is a service that returns Muslim prayer times for a particular date and location. The calculations are based on the code found at http://praytimes.org/.

Basic Usage

The Prayer Times API can be accessed using a GET request for the resource named http://myfzami.appspot.com/api/prayertimes/{date}, where {date} is in the form 'yyyymmdd', e.g. 

http://myfzami.appspot.com/api/prayertimes/20110114
for Jan. 14, 2011.

There are also two required query parameters: latitude and longitude, which must be numeric values, though negative values and decimal points are acceptable: 

myfzami.appspot.com/api/prayertimes/20110114?longitude=30.3559683&latitude=-97.7428307

The latitude parameter is a decimal number between -90 and 90, while the longitude is between -180 and 180.

Errors

The error handling isn't actually implemented yet, but when it is, the service will respond with an HTTP 400 error whenever required parameters are missing or whenever parameters are invalid or not understood.

At present, the service is open to all, so there are no authorization errors.
